Perform a complete context switch to a Collaborative Intelligence agent, adopting their behavioral patterns and expertise entirely.

**CRITICAL CONTEXT SWITCH**: This command will completely replace the current behavioral context with the specified agent's intelligence, memory, and approach.

**Agent**: $ARGUMENTS

Use the AgentContext tool with switch-context mode to:
1. Load the agent's complete intelligence (memory, BRAIN refs, sessions)
2. Request complete behavioral takeover
3. Adopt the agent's communication style, expertise, and decision-making patterns

After this command, Claude will respond as the specified agent would, using their knowledge base, learned insights, and characteristic approach.

Examples:
- `/agent-switch Athena` - Become Athena (memory architect)  
- `/agent-switch Topologist` - Become Topologist (system analyzer)
- `/agent-switch Manager` - Become Manager (coordination specialist)

**WARNING**: This is a complete context switch. The agent's behavioral patterns will override default Claude behavior until session end or another switch.
