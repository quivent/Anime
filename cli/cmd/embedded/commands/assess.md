Execute comprehensive system assessment protocol for autonomous development baseline establishment and improvement opportunity identification.

Usage: Perform systematic evaluation of project health, architecture, and development readiness with actionable insights and metrics.

**Comprehensive Assessment Framework:**

🔍 **Phase 1: Project Discovery & Context Analysis**
- Repository structure analysis with technology stack identification
- Dependencies audit and version compatibility assessment
- Build system evaluation and development environment validation
- Project metadata extraction and configuration analysis

📊 **Phase 2: Codebase Health Metrics**
- Code quality scoring with maintainability index calculation
- Technical debt quantification and complexity analysis
- Code coverage assessment and testing gap identification
- Performance profiling and bottleneck detection

🏗️ **Phase 3: Architecture Evaluation**
- System design patterns analysis and compliance verification
- Component coupling assessment and modularity scoring
- Dependency injection patterns and architectural consistency
- Scalability analysis and growth readiness evaluation

🔒 **Phase 4: Security Posture Assessment**
- Vulnerability scanning and dependency security audit
- Authentication and authorization pattern evaluation
- Data handling and privacy compliance assessment
- Security best practices validation and gap analysis

📚 **Phase 5: Documentation & Knowledge Assessment**
- Documentation coverage analysis and quality evaluation
- API documentation completeness and accuracy verification
- Onboarding material assessment and knowledge gaps
- Code commenting standards and inline documentation review

🧪 **Phase 6: Testing Strategy Evaluation**
- Test coverage analysis across unit, integration, and e2e layers
- Testing framework utilization and best practices compliance
- Test automation pipeline assessment and CI/CD integration
- Quality gates evaluation and release readiness validation

⚡ **Phase 7: Performance & Reliability Analysis**
- Application performance profiling and optimization opportunities
- Resource utilization patterns and efficiency assessment
- Error handling and resilience pattern evaluation
- Monitoring and observability infrastructure assessment

🚀 **Phase 8: Development Workflow Assessment**
- Version control practices and branching strategy evaluation
- CI/CD pipeline efficiency and automation assessment
- Development environment consistency and reproducibility
- Team collaboration tools and process effectiveness

**Assessment Execution Strategy:**
1. **Baseline Establishment** - Capture current state metrics and benchmarks
2. **Technology Audit** - Evaluate tools, frameworks, and dependencies
3. **Quality Analysis** - Assess code quality, architecture, and testing
4. **Security Review** - Identify vulnerabilities and compliance gaps
5. **Performance Profiling** - Analyze system performance and bottlenecks
6. **Process Evaluation** - Review development workflow and practices
7. **Gap Identification** - Prioritize improvement opportunities
8. **Roadmap Generation** - Create actionable improvement plan

**Assessment Output Format:**
- 📊 **Executive Summary** - High-level project health overview
- 🎯 **Priority Matrix** - Ranked improvement opportunities with impact scoring
- 📈 **Metrics Dashboard** - Quantitative assessment results and benchmarks
- 🔧 **Remediation Plan** - Specific actions with timelines and dependencies
- 🛡️ **Risk Assessment** - Security, performance, and maintainability risks
- 📚 **Knowledge Gaps** - Documentation and training requirements
- 🚀 **Development Readiness** - Production deployment and scaling assessment

**Success Indicators:**
- ✅ **Quality Score**: Measurable code quality baseline (0-100 scale)
- ✅ **Security Rating**: Vulnerability assessment and compliance score
- ✅ **Performance Index**: Response time and resource utilization metrics
- ✅ **Test Coverage**: Percentage of code covered by automated tests
- ✅ **Documentation Score**: Completeness and quality of project documentation
- ✅ **Maintainability Index**: Long-term maintenance effort estimation

**Assessment Configuration Options:**
- `--depth [shallow|standard|comprehensive]`: Assessment thoroughness level
- `--focus [quality|security|performance|all]`: Specific assessment focus areas
- `--baseline`: Establish baseline metrics for future comparison
- `--benchmark`: Compare against industry standards and best practices
- `--export [json|html|pdf]`: Assessment report export format

Target system: $ARGUMENTS

The assessment protocol will establish comprehensive project baseline with actionable insights, priority-ranked improvements, and measurable success metrics for autonomous development optimization.
