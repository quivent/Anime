## Usage Examples

```bash
# Compare two files
differentiate file1.txt file2.txt

# Compare agents with metrics
differentiate @analyst @researcher --metrics

# Compare projects with detailed output
differentiate project-a/ project-b/ --detailed --export=report.md

# Compare commands with performance analysis
differentiate "npm install" "yarn install" --performance

# Compare with custom output format
differentiate entity1 entity2 --format=json --output=comparison.json
```

## Command Options

- `--metrics`: Include quantitative metrics when applicable
- `--detailed`: Generate comprehensive detailed analysis
- `--format=[markdown|json|html|pdf]`: Output format selection
- `--export=<filename>`: Export results to specified file
- `--performance`: Include performance benchmarking
- `--verbose`: Detailed progress and diagnostic information
