Execute comprehensive numerical validation and statistical accuracy verification for all displayed data, metrics, and quantitative claims.

Usage: Systematically validate all statistics, calculations, counts, percentages, and numerical information for accuracy and consistency across documents, interfaces, and reports.

**Sequential Fact Validation Protocol:**

🔢 **Phase 1: Numerical Data Extraction**
- Parse all documents, interfaces, and reports for quantitative claims
- Extract statistics, percentages, counts, measurements, and calculations
- Identify data sources and claimed methodologies
- Catalog all numerical assertions requiring verification

📊 **Phase 2: Statistical Consistency Analysis**
- Cross-reference related statistics for mathematical consistency
- Verify percentage calculations against reported totals
- Check that component sums equal reported aggregates
- Identify contradictory numerical claims across sources

🧮 **Phase 3: Mathematical Validation**
- Recalculate all derived statistics and percentages
- Verify formula applications and mathematical operations
- Check unit consistency and measurement accuracy
- Validate statistical methodology and sampling claims

🔍 **Phase 4: Source Data Verification**
- Trace statistics back to original data sources
- Verify file counts, size measurements, and system metrics
- Cross-check against actual system state and configurations
- Validate database queries and data aggregation methods

⚖️ **Phase 5: Comparative Accuracy Assessment**
- Compare claimed statistics against independently measured values
- Identify discrepancies and quantify error margins
- Assess significance of statistical variations
- Validate time-series data and trend claims

📈 **Phase 6: Contextual Validation**
- Verify statistics make logical sense in context
- Check for impossible values or unrealistic claims
- Validate order-of-magnitude appropriateness
- Assess statistical significance and confidence levels

🎯 **Phase 7: Comprehensive Accuracy Report**
- Generate detailed validation results with evidence
- Provide corrected values for inaccurate statistics
- Rate overall numerical reliability and accuracy
- Recommend specific corrections and improvements
