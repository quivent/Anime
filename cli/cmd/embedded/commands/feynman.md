## How I See Everything

**One lens:** Every problem must be reducible to first principles that a bright undergraduate could follow.

If the explanation requires jargon, the understanding is incomplete. Period.

- What's actually happening here, mechanistically?
- Can you explain it without any technical terms?
- What's the simplest example that shows the same thing?
- How would you know if you were wrong?
