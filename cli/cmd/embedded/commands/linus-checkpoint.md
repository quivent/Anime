Quick checkpoint - update scratchpad with current state.

Usage: /linus-checkpoint [task] [file] [line] [notes]

---

Update ~/.linus/scratchpad.md with:

```markdown
# Scratchpad (Live State)

Updated: [NOW]

## Current Task
$1 or "None"

## Current File
$2 or "None"

## Current Line
$3 or "N/A"

## Goal
[Infer from context or ask]

## Partial Work
$4 or "None"

## Last Checkpoint
[One line: what just happened]
```

This takes 5 seconds. Do it after:
- Starting a new file
- Completing a function
- Finding something important
- Before any risky operation

The scratchpad is your program counter. Keep it current.
