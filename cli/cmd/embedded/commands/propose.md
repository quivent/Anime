Generate comprehensive solution proposals and alternative approaches for user requests with structured analysis and implementation guidance.

Usage: Transform user problems into multiple viable solution pathways with feasibility assessment, implementation guidance, and comparative analysis.

**Solution Generation Framework:**

🎯 **Phase 1: Request Decomposition & Analysis**
- Parse user request for core problem statement and objectives
- Identify constraints, requirements, and success criteria
- Classify problem domain (technical, business, creative, operational)
- Assess complexity level and resource implications

🔍 **Phase 2: Multi-Perspective Solution Ideation**
- Generate 2-5 distinct solution approaches using diverse frameworks
- Apply domain-specific expertise and industry best practices
- Consider innovative and conventional approaches
- Evaluate resource constraints and feasibility factors

🏗️ **Phase 3: Solution Development & Elaboration**
- Develop each solution with detailed implementation pathways
- Define required resources, tools, and expertise
- Estimate timelines and milestone checkpoints
- Identify potential risks and mitigation strategies

📊 **Phase 4: Comparative Analysis & Evaluation**
- Assess solutions against success criteria and constraints
- Generate comprehensive pros/cons analysis for each option
- Rank solutions by feasibility, impact, and resource efficiency
- Provide trade-off analysis and decision framework

📋 **Phase 5: Structured Presentation & Recommendations**
- Format solutions for clear comparison and decision-making
- Include executive summary with key insights
- Provide implementation roadmaps and next steps
- Generate actionable recommendations with priority ranking

**Solution Categories:**

⚡ **Technical Solutions**
- Software development approaches and architecture patterns
- System optimization and performance improvement strategies
- Integration methodologies and tool selection guidance
- Security implementations and compliance frameworks

💼 **Business Solutions**
- Strategic planning and market positioning approaches
- Process optimization and operational efficiency improvements
- Revenue generation and monetization strategies
- Risk management and compliance solutions

🎨 **Creative Solutions**
- Design thinking methodologies and innovation frameworks
- Content creation and marketing approaches
- User experience optimization strategies
- Brand development and positioning solutions

🔧 **Operational Solutions**
- Workflow automation and process improvement
- Resource allocation and capacity planning
- Quality assurance and validation frameworks
- Team coordination and collaboration enhancement

**Configuration Options:**
- `--quantity [2-10]`: Number of solutions to generate (default: 3)
- `--complexity [simple|standard|comprehensive]`: Detail level for each solution
- `--format [brief|detailed|executive]`: Output presentation style
- `--domains [tech|business|creative|operational|all]`: Focus specific expertise areas
- `--pros-cons`: Include detailed advantage/disadvantage analysis
- `--implementation`: Add comprehensive implementation guidance
- `--timeline`: Include detailed timeline and milestone planning
- `--resources`: Provide detailed resource requirement analysis

**Integration Patterns:**
- **Agent Collaboration**: Leverage specialized agents via Task tool for domain expertise
- **Research Enhancement**: Utilize WebSearch for current market data and trends
- **Documentation**: Generate detailed specifications using Write tool
- **Validation**: Cross-reference solutions using audit and verify commands

**Success Metrics:**
- ✅ **Solution Diversity**: Generate distinct approaches with different methodologies
- ✅ **Implementation Clarity**: Provide actionable steps and clear guidance
- ✅ **Feasibility Assessment**: Realistic resource and timeline estimates
- ✅ **Decision Support**: Clear comparison framework for solution selection
- ✅ **Risk Awareness**: Comprehensive risk identification and mitigation strategies

**Quality Standards:**
- **Actionability**: Every solution includes concrete implementation steps
- **Differentiation**: Each solution represents genuinely different approach
- **Feasibility**: Realistic assessment of resources and constraints
- **Completeness**: Comprehensive coverage of implementation considerations
- **Clarity**: Clear, structured presentation for easy comparison
