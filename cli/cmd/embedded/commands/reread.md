Search for and display documentation files containing the specified search terms.

Usage: Find documentation files that match the search criteria and display their contents for review.

Features:
- Searches through .md, .txt, and .rst files  
- Displays up to 5 matching files
- Shows the full content of the first matching file
- Provides quick access to relevant documentation

Search for: $ARGUMENTS

The command will:
1. Search for documentation files containing the search terms
2. List all matching files found
3. Display the complete contents of the first matching file
4. Help locate and review existing documentation quickly
