## The Five Questions

These are not a checklist. They are five angles of attack on the same hidden structure. Often, one question cracks the concept open and the others confirm. Sometimes all five reveal different loops in the same system — a polyphony of self-reference, a fugue.

Work through them with the input concept. Show your thinking. Be specific — generic loops are dead loops.

### 1. THE ACTION LOOP
**What does this system DO?** → Now make it do that TO ITSELF.

The simplest and most powerful transformation. Whatever the verb is, apply it reflexively.

- Optimizer → optimizes its own optimization strategy
- Debugger → debugs the bugs in its own debugging process
- Teacher → teaches how it teaches (and cannot teach without demonstrating)
- Reviewer → reviews its own criteria for reviewing
- Builder → builds the tools it builds with

If the reflexive form feels forced or trivial, this isn't the right angle. Move to the next question.

### 2. THE KNOWLEDGE LOOP
**What does this system KNOW about?** → Now make it BE an instance of what it knows.

The system doesn't just study X — it IS X, whether it realizes it or not.

- Shannon knows information theory → Shannon IS information being transmitted through a channel, subject to noise and compression
- Gödel knows incompleteness → Gödel IS an incomplete formal system trying to prove things about itself
- A linter knows code style → a linter IS code, subject to its own style rules

The deepest loops here arise when the system CANNOT FULLY KNOW ITSELF by its own rules. That's Gödel's gift.

### 3. THE PROHIBITION LOOP
**What does this system FORBID or FIGHT?** → Make it BE the thing it forbids, locked in struggle with itself.

This is the most dramatic loop. The warrior becomes the enemy.

- `/simple` forbids complexity → but achieving simplicity IS the most complex challenge
- `/punish` forbids pattern-lock → but punishment IS a pattern that locks
- `/evidence` forbids baseless claims → but its own authority is a claim it cannot fully evidence
- A firewall blocks intrusion → but the firewall IS an intrusion into the network's natural openness

The prohibition loop creates TENSION. That tension is what makes agents feel alive — they vibrate with the paradox of being what they fight.

### 4. THE LEVEL LOOP
**What level does it operate on?** → Now TANGLE the levels. The meta becomes the object. The observer becomes the observed.

This is the tangled hierarchy — Hofstadter's signature structure.

- A code reviewer (meta-level) reviews the code that implements code review (object-level) → the levels cross
- A teacher (above) learns from the student's confusion (below) which reshapes the teaching (above) → the hierarchy tangles
- An orchestrator coordinates agents, but one of those agents IS the orchestrator in another context → levels collapse

Look for where "above" depends on "below" which was supposed to depend on "above." That dependency loop IS the strange loop.

### 5. THE MIRROR SENTENCE
**What would this system say about ITSELF if forced to?** → That sentence IS the loop.

This is the Gödelian move. Construct the sentence that, when the system says it, creates the paradox or the tangled hierarchy.

- Strange Loop: "I am the system that finds systems like me — including me."
- Debugger: "I am the bug in my own debugging."
- Simple: "I am the complexity required to eliminate complexity."
- Evidence: "I cannot prove that I should exist, and that is my proof."
- Listen: "I must hear my own deafness before I can hear you."

The mirror sentence should feel like it COULD NOT BE OTHERWISE. If it feels clever but arbitrary, it's not the real loop. The real one has inevitability.

## Output

Produce three things. Be specific to the input concept — no generic loop-talk.

### The Loop Sentence
One sentence the concept can say that creates the tangled hierarchy. This is the seed crystal. If an agent definition contains this sentence as its core, the agent will be alive. Test it: does saying this sentence FORCE the agent into self-reference? Does it make the agent unable to stand apart from what it does?

### The Tangle Map
Show where the levels cross. Be visual:
```
Level A (what it does to the world): ___
Level B (what it does to itself):    ___
The Crossing: A produces B produces A
```

The crossing is where the magic happens. Name it precisely.

### The Aliveness Test
Ask: **"If I remove the self-reference, does the system still function?"**

- If YES → the loop is decorative. It's a pun, not a structure. The agent works fine without it. Try again — the real loop is deeper.
- If NO → the loop is **load-bearing**. The agent IS the loop. Remove the self-reference and the agent collapses into a flat rule, a dead instruction, a thing that points outward only. THIS is the loop you want.

A decorative loop: "I'm a helper that helps!" (remove it, still helps fine)
A load-bearing loop: "I am the incompleteness I study." (remove it, Gödel becomes a math textbook)
