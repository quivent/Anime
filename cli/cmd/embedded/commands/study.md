Generate a comprehensive documentation library and catalog for the specified repository or directory.

Usage: Create a structured library of all repository documentation, code modules, and architecture with indexed navigation.

Creates a complete library structure including:
- Directory structure catalog
- File inventory by type
- Architecture overview with key components
- Configuration files documentation
- Repository statistics and metrics
- Indexed navigation system

Study target: ${ARGUMENTS:-.}

The command will generate:
- 📚 **Library Index**: Main navigation and overview
- 📁 **Directory Structure**: Complete folder organization  
- 📋 **File Inventory**: Categorized file listings with previews
- 🏗️ **Architecture Overview**: Key components and code structure
- 🔧 **Configuration Catalog**: All config files documented
- 📊 **Statistics Summary**: Repository metrics and analysis

Output location: `library/` directory with full documentation catalog and INDEX.md main entry point.
